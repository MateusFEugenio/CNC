### CNC
  CNC (Computer Numeric Control ou Controlo numérico computadorizado) é um sistema que permite o controle da máquina.

  Tipos de máquinas que são variações da CNC:
  * Fresadora
  * Torno
  * Jato de água
  * Retificadora
